QFold на Amazon Braket

Краткое руководство по развертыванию квантового алгоритма сворачивания белка.

1
Доступ к консоли: Войдите в свою учетную запись AWS и перейдите в сервис Amazon Braket. Убедитесь, что вы находитесь в регионе, поддерживающем нужные устройства (например, us-east-1).
2
Создание блокнота: В левом меню выберите Notebooks -> Create notebook instance.
  • Имя: qfold-instance
  • Тип: ml.t3.medium (минимально достаточный)
3
Открытие Jupyter: Когда статус сменится на InService, нажмите Open JupyterLab.
4
Поиск QFold: Перейдите по пути: amazon-braket-examples / examples / hybrid_quantum_algorithms / qfold и откройте файл qfold.ipynb.
5
Настройка устройства: В коде найдите переменную device.
  • Для бесплатного теста: LocalSimulator()
  • Для QPU: укажите ARN (например, Rigetti или IonQ).
Внимание: Запуск на реальных квантовых процессорах (QPU) платный. Сначала отладьте код на локальном симуляторе.
6
Запуск: Нажмите Run -> Run All Cells. Результаты оптимизации структуры белка появятся в нижних ячейках.
Создано для быстрого старта с QFold в экосистеме AWS.
